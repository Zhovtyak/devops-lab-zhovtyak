# Проекты

Ниже представлены проекты, в которых я разрабатывал backend-системы, Telegram-ботов, AI-интеграции и инфраструктуру с использованием Docker, CI/CD и мониторинга.

## Портфолио проектов

### Проект 1: Uniset — AI Telegram Platform

> **Описание:** Платформа для генерации контента (текст, изображения, видео) через Telegram-ботов с интеграцией нескольких AI-провайдеров. Реализована система токенов, рефералов и масштабируемая backend-архитектура.  
> **Технологии:** Python (asyncio, aiogram), PostgreSQL, Redis, Docker, REST API.  
> **Особенности:** Асинхронная обработка задач, интеграции с AI API, биллинг, rate limiting.

---

### Проект 2: AI Media Generation API

> **Описание:** Набор асинхронных API-клиентов для генерации видео и аудио (text2video, image2video, text2audio) с обработкой очередей и ошибок провайдеров.  
> **Технологии:** Python, aiohttp, asyncio, REST API.  
> **Особенности:** Унифицированный интерфейс для разных AI-провайдеров, retry и fallback логика.

---

### Проект 3: Telegram Mini App (Frontend)

> **Описание:** Веб-приложение внутри Telegram (TMA) для взаимодействия с AI-сервисами: генерация, просмотр истории, управление контентом.  
> **Технологии:** React, TypeScript, Vite.  
> **Особенности:** Интеграция с Telegram Web Apps, работа с API, оптимизация UI под мобильные устройства.
